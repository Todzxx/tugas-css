Cara Mewarnai Teks dengan CSS

Properti color

Properti color digunakan untuk mengatur warna teks pada elemen HTML.

Ada beberapa cara untuk menentukan nilai warna:

1. Named Colors (Nama Warna)

Ini adalah teks berwarna merah menggunakan nama warna "red".

Ini adalah teks berwarna biru menggunakan nama warna "blue".

Ini adalah teks berwarna hijau menggunakan nama warna "green".

2. HEX Values (Nilai Heksadesimal)

Ini adalah teks berwarna ungu menggunakan nilai HEX #800080.

Ini adalah teks berwarna teal menggunakan nilai HEX #008080.

Ini adalah teks berwarna oranye menggunakan nilai HEX #FFA500.

3. RGB Values (Nilai Red, Green, Blue)

Ini adalah teks berwarna coklat menggunakan nilai RGB rgb(165, 42, 42).

Ini adalah teks berwarna biru langit menggunakan nilai RGB rgb(135, 206, 235).

4. RGBA Values (Nilai Red, Green, Blue, Alpha)

Ini adalah teks berwarna hitam dengan opasitas 50% menggunakan RGBA rgba(0, 0, 0, 0.5).

Ini adalah teks berwarna hijau dengan opasitas 70% menggunakan RGBA rgba(0, 128, 0, 0.7).

5. HSL Values (Nilai Hue, Saturation, Lightness)

Ini adalah teks berwarna magenta menggunakan HSL hsl(300, 100%, 50%).

Ini adalah teks berwarna kuning menggunakan HSL hsl(60, 100%, 50%).

6. HSLA Values (Nilai Hue, Saturation, Lightness, Alpha)

Ini adalah teks berwarna cyan dengan opasitas 60% menggunakan HSLA hsla(180, 100%, 50%, 0.6).

Ini adalah teks berwarna abu-abu dengan opasitas 80% menggunakan HSLA hsla(0, 0%, 50%, 0.8).
