Memahami 6 Jenis Selektor di CSS

1. Universal Selector (*)

Memilih semua elemen HTML di halaman.

Ini adalah paragraf di dalam div.

Ini adalah span di dalam div.

2. Type Selector (Elemen)

Memilih semua elemen HTML dengan nama tag tertentu.

Contoh: p akan memilih semua elemen paragraf.

Ini adalah elemen span.

3. Class Selector (.nama-kelas)

Memilih semua elemen HTML dengan atribut class tertentu.

Ini adalah paragraf dengan kelas "my-class".

Ini adalah div dengan kelas "my-class".

4. ID Selector (#nama-id)

Memilih elemen HTML dengan atribut id tertentu. ID harus unik dalam satu halaman.

Ini adalah paragraf dengan ID "unique-id".

5. Group Selector (selector1, selector2)

Memilih beberapa elemen HTML yang berbeda dan menerapkan gaya yang sama.

Ini adalah H3

Ini adalah H4

6. Descendant Selector (ancestor descendant)

Memilih elemen yang merupakan turunan dari elemen lain.

Ini adalah paragraf di dalam elemen dengan kelas "parent".

Ini adalah span di dalam elemen dengan kelas "parent".
