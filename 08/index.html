Pengaturan Font di CSS

Ini adalah contoh teks dengan properti font-family. Kita bisa menentukan beberapa font sebagai fallback jika font pertama tidak tersedia.

Ukuran font paragraf ini diatur menggunakan properti font-size. Kita bisa menggunakan unit seperti px, em, rem, atau persentase.

Berat font (ketebalan) teks ini diatur dengan font-weight. Nilai yang umum adalah normal, bold, atau angka dari 100 hingga 900.

Gaya font teks ini diatur menggunakan font-style. Nilai yang umum adalah normal, italic, atau oblique.

Transformasi teks ini diatur dengan text-transform. Bisa uppercase, lowercase, atau capitalize.

Tinggi baris teks ini diatur dengan line-height. Ini mengontrol jarak vertikal antar baris teks.

Jarak antar huruf teks ini diatur dengan letter-spacing. Ini mengontrol spasi horizontal antar karakter.

Jarak antar kata teks ini diatur dengan word-spacing. Ini mengontrol spasi horizontal antar kata.

Ini adalah contoh penggunaan properti font shorthand. Ini memungkinkan kita mengatur beberapa properti font dalam satu deklarasi.
