Sintaks Dasar CSS yang Harus Dipahami

Selector

Selector digunakan untuk "menemukan" atau memilih elemen HTML yang ingin diberi gaya.

Contoh: p memilih semua elemen paragraf.

Ini adalah paragraf dengan kelas "highlight".

Ini adalah paragraf dengan ID "unique".

Deklarasi (Property dan Value)

Setelah selector, ada blok deklarasi yang berisi satu atau lebih deklarasi CSS.

Setiap deklarasi terdiri dari properti CSS dan nilai, dipisahkan oleh titik dua (:).

Contoh: color: blue; (properti: color, nilai: blue)

Deklarasi dipisahkan oleh titik koma (;).

Struktur Lengkap


selector {
  property: value;
  property: value;
}

Contoh:


p {
  color: red;
  font-size: 16px;
}

.highlight {
  background-color: yellow;
}

#unique {
  border: 1px solid green;
}
