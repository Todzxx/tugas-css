3 Cara Menuliskan CSS di HTML

1. Inline Style

Ini adalah contoh menggunakan inline style. Atribut style ditambahkan langsung pada elemen HTML.

2. Internal/Embedded Style

Ini adalah contoh menggunakan internal style. Kode CSS ditulis di dalam tag <style> di bagian <head> dokumen HTML.

3. External Style Sheet

Ini adalah contoh menggunakan external style sheet. Kode CSS ditulis di file terpisah (misal: style.css) dan dihubungkan ke file HTML menggunakan tag <link>.
